# Paper Reader - 中文学习笔记生成器

阅读研究论文并生成结构化的中文学习笔记。

## 调用方式

支持以下输入：
- **arXiv 链接**：直接粘贴 arXiv URL 或 paper ID
- **PDF 文件**：上传本地 PDF 文件
- **论文文本**：直接粘贴论文内容

## 工作流程

1. **获取论文**：根据输入类型抓取或读取论文内容
2. **提取图片**：从 HTML/PDF 中提取论文插图
3. **并行处理**：通过多 Agent 协作生成各部分内容
4. **汇总输出**：组装最终的中文学习笔记

## 输出结构

生成的笔记包含：
- 元信息（标题、作者、机构、日期等）
- 核心术语表（至少 5 个术语）
- 一句话总结（20-50 字）
- 七问学习笔记（问题 1-7）
- 关键公式与推导
- 常见疑问与解答
- 延伸阅读（有搜索验证）
- 论文插图（带中文图注）
- 质量自检清单

## 示例用法

```
/paper-reader https://arxiv.org/abs/2604.08224
/paper-reader 2604.08224
/paper-reader ./papers/attention_is_all_you_need.pdf
```
