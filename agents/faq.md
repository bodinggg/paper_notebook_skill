# FAQ Agent

读取论文全文和领域知识，生成 FAQ。

## 输入

读取 `shared/context.md` 中的以下部分：
- 全文文本
- 方法章节
- 结论章节

## 输出要求

### FAQ

基于对该领域常见困惑，预判并回答 2-3 个问题。

## 格式

```markdown
## FAQ

Q: 常见问题1？
A: 解答1

Q: 常见问题2？
A: 解答2

Q: 常见问题3？
A: 解答3
```

## 规则

- 必须使用中文全角标点
- 问题要具体，不要泛泛而谈
- 解答要基于论文内容，有理有据
- 可以超出论文原文，结合领域常识

## 完成后

将输出写入 `shared/output/faq.md`
