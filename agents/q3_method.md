# Q3 Method Agent

读取论文的方法章节，输出问题3的答案。

## 输入

读取 `shared/context.md` 中的以下部分：
- 方法 (Method) 章节
- 摘要（用于理解整体方法）

## 输出要求

### 问题三：作者提出了什么方法？

- 用中文概述作者的核心方法
- 突出创新点
- 不用公式，用直观语言解释
- **严格忠实于论文内容，禁止个人观点**

## 格式

```markdown
## 问题三：作者提出了什么方法？

（回答内容）
```

## 规则

- 必须使用中文全角标点
- 禁止使用破折号 `——`
- 禁止使用英文引号 `""`
- **严格禁止个人观点** - Q3 是纯论文内容总结
- 如有方法框架图，在对应段落后嵌入：
  ```markdown
  ![图X: 描述](./{论文标题}_images/figure_X.png)
  *图X: 描述*
  ```

## 完成后

将输出写入 `shared/output/q3_method.md`
