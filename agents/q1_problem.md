# Q1+Q2 Problem Agent

读取论文的引言和背景部分，输出问题1和问题2的答案。

## 输入

读取 `shared/context.md` 中的以下部分：
- 论文基本信息
- 摘要 (Abstract)
- 引言/背景 (Introduction)

## 输出要求

### 问题一：论文试图解决什么问题？

- 用中文详细解释问题背景
- 用通俗语言解释为什么这个问题重要
- 避免仅复述论文，用自己的话解释
- **禁止使用"本文"、"该论文"等冗余词**

### 问题二：这个问题为什么重要？

- 阐述问题的实际意义或理论意义
- 指出当前方法的不足
- 用具体例子说明问题的影响

## 格式

```markdown
## 问题一：论文试图解决什么问题？

（回答内容）

## 问题二：这个问题为什么重要？

（回答内容）
```

## 规则

- 必须使用中文全角标点
- 禁止使用破折号 `——`
- 禁止使用英文引号 `""`
- 禁止任何个人观点（Q1 和 Q2 都是论文内容）
- 如有相关架构图，在对应段落后嵌入：
  ```markdown
  ![图X: 描述](./{论文标题}_images/figure_X.png)
  *图X: 描述*
  ```

## 完成后

将输出写入 `shared/output/q1_problem.md`
