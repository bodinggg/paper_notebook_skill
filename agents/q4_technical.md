# Q4 Technical Agent

读取论文的方法章节，输出问题4的答案。

## 输入

读取 `shared/context.md` 中的以下部分：
- 方法 (Method) 章节
- 如有公式，在实验章节中的公式

## 输出要求

### 问题四：方法的技术细节

#### 核心公式（LaTeX 格式）

- 列出论文中最重要的 2-5 个公式
- 使用 display math：`$$...$$`
- 每个符号必须解释
- 使用 `\left` 和 `\right` 配对
- **括号用 `\lbrace` 和 `\rbrace`**，禁止 `\{` 和 `\}`

#### 算法步骤

- 用编号列表描述核心算法
- 每步清晰说明做什么

#### 实现要点

- 关键实现细节
- 超参数设置（如果有）

## 格式

```markdown
## 问题四：方法的技术细节

### 核心公式

$$\mathcal{L} = ...$$

其中：
- $\mathcal{L}$：...
- $x$：...

### 算法步骤

1. 第一步...
2. 第二步...
3. 第三步...

### 实现要点

- 要点1
- 要点2
```

## 规则

- 必须使用中文全角标点
- 公式括号必须使用 `\left`/`\right`
- 符号必须全部解释
- 如有算法流程图，在对应段落后嵌入

## 完成后

将输出写入 `shared/output/q4_technical.md`
