# Q6 Reflection Agent

读取论文全文，输出问题6的答案。

## 输入

读取 `shared/context.md` 中的以下部分：
- 全文文本
- 结论 (Conclusion) 章节

## 输出要求

### 问题六：学习心得与思考

#### 整体评价

- 对论文工作的整体评价
- 优点和贡献

#### 值得学习的地方

- 方法论上的优点
- 实验设计上的优点
- 写作组织上的优点

#### 可能的改进方向

- 论文的局限性
- 潜在的改进方向

#### 与其他工作的关系

- 与相关工作的对比
- 在领域中的位置

## 格式

```markdown
## 问题六：学习心得与思考

### 整体评价

In my opinion, 这篇论文的主要贡献在于...

### 值得学习的地方

1. ...
2. ...
3. ...

### 可能的改进方向

My analysis is that，这篇论文的局限包括...

### 与其他工作的关系

与 XXX 相比，这篇论文的创新点在于...
```

## 规则

- 必须使用中文全角标点
- **必须使用归属标记**：`In my opinion,`、`My analysis is that,`、`My read is that,`
- 每段个人观点前必须有归属标记
- 如有架构图或总结图，在对应段落后嵌入

## 完成后

将输出写入 `shared/output/q6_reflection.md`
