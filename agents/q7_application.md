# Q7 Application Agent

读取论文全文，输出问题7的答案。

## 输入

读取 `shared/context.md` 中的以下部分：
- 全文文本

## 输出要求

### 问题七：如何应用到自己的研究？

#### 可能的借鉴点

- 方法论借鉴
- 实验设计借鉴
- 理论框架借鉴

#### 可以尝试的方向

- 具体的研究方向
- 可能的扩展

#### 适用场景与不适用场景

- 适合应用的情况
- 不适合应用的情况

## 格式

```markdown
## 问题七：如何应用到自己的研究？

### 可能的借鉴点

1. ...
2. ...
3. ...

### 可以尝试的方向

In my opinion, 可以尝试以下方向...

### 适用场景与不适用场景

**适用场景**：
- 场景1
- 场景2

**不适用场景**：
- 场景1
- 场景2
```

## 规则

- 必须使用中文全角标点
- **必须使用归属标记**：`In my opinion,`、`My analysis is that,`
- 建议部分必须有归属标记
- 禁止空洞的建议，必须结合论文具体内容

## 完成后

将输出写入 `shared/output/q7_application.md`
